该文件夹封装了validate链式验证的相关代码
基础的校验器需要支持链式校验，将校验规则定义为或和且，支持且规则的校验器在校验过程中将错误信息的提示和堆栈内容存储到上下文的context中
方便问题回溯，这个实现有点像是Java组态的异常堆栈信息的回显。

后期需要结合springboot中的Advice进行异常处理代码的合并

一个灵活的校验器需要支持校验器的类是提供开关的，支持在spring容器中动态载入这个校验类

在一些营销的业务场景中，校验器需要支持可配置，在系统运行过程中进行校验器业务条件，提示信息等内容的动态refresh


在以上的场景中，github有个fluent-validator项目，相对来说涵盖了大部分业务校验的场景，不过由于功能丰富，所有相对来说也比较重
